매뉴얼의 계층 경로를 통째로 보존한 덕분에, 검색이 정확해지고("어느 화면의 어느 항목") 답변의 출처가 사람이 검증 가능한 형태로 남습니다. 이 구조가 이 시스템의 차별점입니다.

기술 스택 (전부 오픈소스)
한국어 임베딩 ko-sroberta 리랭커 BGE-reranker FAISS 벡터 검색 BM25 키워드 검색 로컬 LLM(Ollama) · 발췌 폴백 Python 표준 웹서버

LLM이 내려가 있어도 근거 발췌 모드로 답변이 계속되는 무중단 폴백 구조이며, 모델 교체(고정밀 임베딩 등)는 환경변수 변경만으로 가능합니다.
