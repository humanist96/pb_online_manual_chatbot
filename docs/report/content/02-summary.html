어떻게 — 매뉴얼 원문의 계층 구조를 그대로 보존해 검색 인덱스로 만들고(브레드크럼 방식), 의미 검색과 키워드 검색을 결합한 하이브리드 방식으로 찾아, 검색된 근거 안에서만 답변을 생성합니다. 매뉴얼에 없는 질문은 지어내지 않고 "확인되지 않았어요"라고 답합니다.

왜 자체 구축 — 사내 폐쇄망·데이터 보호 요건상 외부 상용 AI API를 쓸 수 없습니다. 임베딩·검색·답변 생성 전부를 오픈소스 모델로 사내 서버 안에서 해결했고, 덕분에 추가 라이선스 비용도 없습니다.

현재 상태 — 파서 품질은 회귀 테스트 5/5로 검증, Docker·systemd 배포 패키징 완료, 검색 품질을 눈으로 검증하는 QA 계측 콘솔 내장. 소개 영상 2편(보고용 3분·실무 상세본 4분)을 함께 제작했습니다.
